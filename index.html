Git komandos

  • init - sukuria gito repozitorija
  • add - prideda nurodytus failus prie commitui paruostu failu
  • status - nurodo informacija apie tavo lokalios gito repozitorijos statusa
  • commit -m "zinute" - sukuria kodo versija is stageintu failu ir prideda zinute apie pakitimus (zinute bandyti islaikyti ne ilgesne nei 50 simboliu)
  • push - is lokalios repozitorijos commitu nusiuntimas i isorine repozitorija
  • pull - is isorines repozitorijos commitu atsisiuntimas i lokalia repozitorija
  • remote add origin https://... - isorines repozitorijos prijungimas prie lokalios repozitorijos
  • log - cummitu sarasas
